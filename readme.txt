langkah-langkah untuk menjalankan aplikasinya

1. Masuklah ke link streamlit yang sudah di deploy https://kriptografimukhtarmundzir.streamlit.app/
2. Kemudian anda dapat memilih salah atu dari 4 algoritma enkripsi
3. Saat sudah masuk ke menu enkripsi anda bisa memilih apa yang ingin di enkripsi apakah Plaintext atau File
4. Jika sudah di enkripsi kemudian unduh hasilnya
5. Pilih menu dekripsi untuk mendekripsi hasil yang sudah anda unduh tadi
6. Setelah itu pilih jenis file apa yang ingin didekripsi kemudian unduh hasilnya
7. Untuk algoritma DES dan AES terdapat 3 pilihan tambahan yaitu ECB, CBC, CTR
